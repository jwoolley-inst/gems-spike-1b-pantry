Reframe the question from "does it have the features" to "how long do teachers and IT spend keeping those features working." Feature checklists compare tables; teacher hours and IT tickets compare reality. Offer a side-by-side of three common workflows — gradebook update, parent-visible assignment, SSO failure recovery — using the district's own last-month ticket data.
